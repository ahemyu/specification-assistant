# DEPRECATED - Vanilla JS Implementation
## Overview
This directory contains the original vanilla JavaScript implementation of the PDF Reader frontend.

## Contents
- `static/` - Static assets (JS, CSS, images)
- `templates/` - Jinja2 HTML templates

## Why Archived?
The application has been fully migrated to **React + TypeScript + Vite** for better:
- Type safety
- Component reusability
- State management 
- Developer experience
- Maintainability

## Current Implementation
The active frontend is located in:
- `/src/pdf_reader/frontend/src/` - React source code
- `/src/pdf_reader/frontend/dist/` - Production build
